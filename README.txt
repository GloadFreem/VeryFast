金指投项目产品版本管理方案
GitHub地址：https://github.com/GloadFreem/VeryFast.git
1.文档结构
project_server:后台工程（java）
project_android:安卓工程 (java)
project_iOS:iOS工程（Objective-c)
2.提交规则：
 2.1每日必须要求项目成员备份上传代码
 2.2管理员审核代码质量及上传时间
3.代码同步规则：
 3.1只有同组成员可以获取本工程源码，并开设分支进行编码，由管理员统一合并项目进行测试
 3.2不得将不属于自己工作范畴的工程源码获取并进行传播及修改
 3.3版本管理规范化，标准化
